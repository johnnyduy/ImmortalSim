# ImmortalSim

A text-based cultivation roguelite MVP built with Next.js, TypeScript, and Tailwind.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000`

## Features

- Start new simulation
- Random event selection by age
- Player choices with stat effects
- Age progression and death system
- Reincarnation with persistent inheritance
- Simple stats panel and event log UI
- Mobile-friendly minimalist layout
